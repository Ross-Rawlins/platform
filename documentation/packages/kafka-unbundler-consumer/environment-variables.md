# Environment Variables

| Variable Name | Type   | Relevance      | Required | Default |
| ------------- | ------ | -------------- | -------- | ------- |
| KAFKA\_HOST   | String | Kafka hostname | No       | kafka   |
| KAFKA\_PORT   | Number | Kafka port     | No       | 9092    |
